# Beomseong Kim - Portfolio Website

This is the personal portfolio website of Beomseong Kim, a Data Scientist based in Seoul, South Korea. The website showcases my projects, skills, and professional history.

## Design & Tech Stack

- **Design**: Bright, modern, and clean UI with a focus on readability and user experience.
- **Tech Stack**: Pure HTML5, CSS3, and JavaScript (Vanilla).
- **Hosting**: Optimized for GitHub Pages.
- **Responsiveness**: Mobile-first design, fully responsive across all devices.

## Features

- **Hero Section**: Introduction and call to action.
- **About Section**: Professional summary and skill stack.
- **History Section**: Timeline of education and work experience.
- **Portfolio Section**: Filterable grid of projects with detailed modals.
- **Contact**: Social links and email.

## Deployment

This website is designed to be hosted on GitHub Pages.

1.  Push the code to your GitHub repository (`bonniecoleman.github.io` or similar).
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch as the source.
4.  Your site will be live at `https://bonniecoleman.github.io/`.

## Local Development

To run the website locally:

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Alternatively, use a local server like Live Server in VS Code for a better experience.
