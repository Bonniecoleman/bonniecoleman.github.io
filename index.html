Pegasus GUI

Abstract

The Pegasus GUI project is designed to load and analyze electromagnetic data efficiently through a user-friendly PyQt5-based interface. It provides researchers with tools to visualize, calibrate, and process data in real-time, improving the workflow for experimental data analysis.

Main Features

- Load and process raw experimental shot data with customizable configurations.
- Plot data in real-time, including time-series and spectrogram visualizations.
- Manage data categories dynamically for better control and analysis.

Development Environment

- Programming Language: Python
- Libraries Used: PyQt5, Matplotlib, NumPy
- GUI Framework: PyQt5
- Key Functions: Real-time plotting, data calibration, user-configurable data handling

Categorical Feature Encoding Challenge

Resource Code: Link

Abstract

This project explores encoding techniques for categorical features, using Kaggle's "Cat in the Dat" dataset. A Logistic Regression model was trained and optimized, achieving a private leaderboard score of 0.80761.

Main Features

  • Comprehensive encoding for binary, nominal, ordinal, and cyclical features
  • Exploratory Data Analysis (EDA) to analyze feature distributions
  • GridSearchCV for hyperparameter tuning

Development Environment

  • Programming Language: Python
  • IDE: Jupyter Notebook
  • Libraries: scikit-learn, pandas, seaborn, matplotlib
  • Dataset: Kaggle
